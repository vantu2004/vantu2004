## 📌 Featured Project

### Todaii English — Full-Stack AI English Learning Platform

*Full-stack Developer, Team Lead (2 members) · Nov 2025 - Jun 2026*

A multi-module Spring Boot backend with 30+ entities and 230+ RESTful API endpoints powering an English learning platform:

- AI-powered personalized study engine analyzing TOEIC performance to generate study plans
- TOEIC practice & analytics system with score prediction and CEFR mapping
- Spring AI integration with OpenAI/Gemini, including a streaming chatbot with JDBC-persisted chat memory
- Redis-backed dictionary (cache → DB → external APIs)
- Deployed on Render using Docker

**Tech Stack:** Java, Spring Boot, Spring AI, Spring Security, Spring Data JPA, MySQL, Redis, Docker, OpenAI, Gemini, Google Translate API, YouTube Data API v3, NewsAPI, Cloudinary
